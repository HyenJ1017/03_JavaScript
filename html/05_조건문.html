조건문 

    프로그래밍 코드가 해석되는 순서를 제어하는 제어문의 한 종류
    작성된 조건식에 따라서 수행되는 코드를 변경하는 구문

    * 조건식: 결과가 true/ false 인 식
    ex) boolean 단독작성
        비교 연산자, 논리 연산자

if(만약에...) 문

    
      if(조건식) {
        조건식이 tre인 경우 수행 할 코드;
      }

입력받은 수가 양수인지 아닌지 검사

if-else(아니면)문

    
      if(조건식){
        조건식이 true일 경우 수행 할 코드;
      }

      else {
          //else는 조건식을 작성하는 ()가 없다!!!
        조건식이 false인 경우 수행 할 코드;
      }

0 ~ 99 사이의 난수를 발생시켜 홀/짝 판별하기
